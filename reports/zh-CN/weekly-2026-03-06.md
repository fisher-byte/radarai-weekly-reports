## 本周总览
- Google 发布 Gemini 3.1 Flash Image（Nano Banana 2），登顶 Image Arena，首次实现「实时网页搜索 + 多模态理解」双路验证生图，主体一致性与事实可靠性突破金融/舆情等强约束场景。

- Anthropic 因拒绝军事合作遭特朗普政府单方面列为“供应链风险”并下达联邦禁用令，美国财政部、五角大楼全面下线 Claude，但 OpenAI、Google 公开声援，伦理立场正式成为头部厂商核心竞争力与地缘政治变量。

- Claude Code 全面上线自动记忆、语音模式、远程控制及 `/simplify`/`/batch` 双智能体技能，形成首个覆盖「记忆→输入→执行→重构→迁移」全链路的开发者 Agent 生产环境。

## 热点清单
1. Gemini 3.1 Flash Image（Nano Banana 2）正式发布  
https://blog.google/technology/ai/gemini-3-1-flash-image/  
本质：Google 推出首个融合实时网页搜索与 Gemini 多模态理解的极速图像生成模型，支持 4K 超分、8:1 超宽比及「思考层级」配置，在 Image Arena 登顶，关键突破在于通过双路验证机制（Gemini 理解 + 网页事实检索）显著抑制幻觉，使生图结果在金融图表、新闻配图等需强事实性场景具备工程可用性。  
——可能：个人开发者立即用 `curl -X POST "https://generativelanguage.googleapis.com/v1beta/models/gemini-3.1-flash-image:generateContent?key=YOUR_KEY"` 测试带 URL 引用的指令（如“生成 2025 年美联储利率路径图，数据来源：https://www.federalreserve.gov/monetarypolicy/fomcprojtabl20250301.htm”），验证事实对齐能力；产品侧可在财报助手、舆情监测工具中嵌入该模型 API，替代传统爬虫+模板渲染流程。

https://www.samsung.com/global/galaxy/ai/

——可能：开发者立即注册 Samsung Developer Program（https://developer.samsung.com/galaxy/ai），调用 `SamsungAISearchIntent` API 在自有 App 中触发 Perplexity 搜索；产品侧可设计「S26 专属功能开关」，例如在电商 App 中长按商品图直接唤起 Perplexity 对比竞品参数并生成购买建议。
